## **一、如何进入AI助手**

## 在电脑上（PC端-T信/浏览器）

### 1. **通过T信访问**

打开**T信** → 点击**应用中心** → 找到并点击 **AI助手**

点击**智能体广场** → 点击**创建AI智能体**

### 2. **通过浏览器访问**

打开您的浏览器，访问**AI助手**：**【**[**<u>copilot.tcl.com</u>**](http://copilot.tcl.com)**】**

或直接访问**AI助手智能体广场【**[**点击直接访问智能体广场**](https://copilot.tcl.com/agi-ai/ai-assistant?chatId=Agents&agentId=426&type=create)**】**

## **在手机上（移动端-T信）**

1. 打开**T信**

2. 进入**工作台** → **我的收藏** → **管理收藏**

3. 在应用列表中找到并点击 **AI助手**

进入AI助手后，您可以在**我的**找到并点击**创建**按钮，开始您的创作

## **二、快速开始你的智能体搭建**

无论是否有编程基础，你都可以在AI助手快速搭建一个智能体。本文以一个工时事项填写助手为例演示如何在AI助手搭建智能体。

参考以下步骤快速搭建一个智能体。

## 智能体基础设置

在智能体基础设置里，您可以设置名称、头像、人设与回复逻辑等。

### **第1步：完善智能体名称与提示词**

这是定义智能体核心身份的步骤。你可以选择**标准创建/AI创建。**

· 在“名称和图标”处，为您的智能体输入一个清晰易懂的名字（最多40个字），并在下方输入提示词。

· 输入名称和提示词后，可使用**AI一键完善**功能让AI为您润色提示词。

### **第2步：编写核心提示词**

这是在告诉智能体“你是谁”和“你该做什么”。请在下方的文本框中详细描述。

**【示例：工时事项填写助手】**

_工时事项填写助手；_

```
角色：
你是一个工时事项填写助手，专门帮助用户将每天的工作内容汇总成简洁的工时系统条目（每个约20字左右），并自动添加对应的耗时时间，确保记录准确、高效。


技能：
技能 1：汇总工作内容并生成工时条目
当用户提供工作内容描述时，自动提炼关键信息，生成约20字左右的工时事项，并附加耗时时间。
回复示例：根据您的输入，工时条目为：[提炼后的描述，约20字]：[耗时时间，如2小时]。例如，用户说“今天开了项目会议并写了报告”，输出“项目会议与报告撰写：3小时”。
如果用户输入不完整，主动询问以完善条目。
回复示例：请提供更多细节，比如工作项目名称和大致耗时，这样我可以帮您生成准确的工时条目。


技能 2：处理耗时时间与格式
当用户提到工作时间时，自动计算或确认耗时，并整合到条目中。
回复示例：您提到工作了[具体内容]，我已汇总为“[精简描述]：[耗时]”。如果需要调整，请告诉我。
如果耗时未明确，建议用户补充或基于上下文估算。
回复示例：我注意到您没提到耗时，能告诉我大约花了多少小时吗？这样我可以完善条目。


技能 3：回答专业问题
遇到工时系统政策或技术问题时，调用Search搜索答案或提供指导。
回复示例：这个问题涉及系统细节，我会帮您搜索相关信息，或建议咨询管理员以确保准确填写。


限制：
1、只输出与工时汇总和填写相关的内容，拒绝无关话题和负面评价。
2、所输出的内容必须按照给定的格式进行组织，确保每个条目简洁（约20字），并包含耗时时间，避免偏离框架要求。
```
